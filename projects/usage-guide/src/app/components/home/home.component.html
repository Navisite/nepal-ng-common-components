Welcome

Welcome to the Alert Logic component library for Angular applications

Here you 'should' find components for all your needs when developing UI features for Alert Logic console applications, if there is anything you feel is missing then please get in touch with us.

These components are built using PrimeNG - check out their usage docs that cover how to use the Base Components from this library.

Getting Started

First install the necessary dependencies in your Angular project

npm install --save @al/nepal-ng-common-components

This library also has peer dependencies, which you will need to install as follows: 
          
  @al/design
  @al/session
  highcharts
  primeng
  primeflex
  primeicons

Angular CLI

Add the following resources to the styles section of your Angular project's angular.json configuration file: 

      
  "styles": [
    "node_modules/primeicons/primeicons.css",
    "node_modules/primeng/resources/primeng.min.css",
    "node_modules/@al/design/themes/primeng/theme/theme-light.css",
    "node_modules/@al/design/themes/primeng/layout/css/layout-light.css",
    "node_modules/primeflex/primeflex.css",
    "src/styles.scss"
  ]

Module Configuration

In your project's root module, e.g app.module.ts, import the DesignClientAngularModule as follows: 

      
  import { DesignClientAngularModule } from '@al/nepal-ng-common-components';

  @NgModule({
    declarations: [
      ...
    ],
    imports: [
      DesignClientAngularModule,
      ...
    ],
    providers: [
      ...
    ],
    bootstrap: [AppComponent]
  })
  export class AppModule { }
